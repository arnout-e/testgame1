# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Browser-based Tic Tac Toe game. Single `index.html` file with embedded CSS and JS — no build tools, dependencies, or server required.

## Development

Open `index.html` directly in a browser to run the game. No build or test commands.

## Git Workflow

After every wijziging: commit met een duidelijke, beschrijvende commitboodschap en push altijd naar `origin/main` op GitHub (arnout-e/testgame1). Sla geen push over.

## Language

The UI is in Dutch (Nederlands). Keep all user-facing text in Dutch.
